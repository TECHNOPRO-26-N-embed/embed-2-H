# テスト仕様書

- **ファイル名**: `sound_test.ino`
- **機能**: ボタン操作によるLEDのトグル動作と、LED点灯時のブザー音再生

## 3. テスト手順

1. **初期状態確認**
   - 電源を投入し、LEDが消灯していることを確認。
   - ブザーが無音であることを確認。

2. **ボタン操作によるトグル動作**
   - ボタンを1回押下し、LEDが点灯することを確認。
   - シリアルモニタに`LED State: ON`が出力されることを確認。
   - 再度ボタンを押下し、LEDが消灯することを確認。
   - シリアルモニタに`LED State: OFF`が出力されることを確認。

3. **LED点灯時のブザー動作**
   - ボタンを押してLEDを点灯させる。
   - ブザーがドレミファソを300ms間隔でループ再生することを確認。

4. **LED消灯時のブザー停止**
   - ボタンを押してLEDを消灯させる。
   - ブザーが停止し、音が鳴らないことを確認。

5. **ドレミファソのループ再生**
   - LED点灯中に時間経過を観察。
   - ブザーがドレミファソを順番に再生することを確認。

6. **チャタリング防止確認**
   - ボタンを短時間で連続押下。
   - LEDの状態が1回の押下ごとに正しく切り替わることを確認。

## 4. テスト環境
- **ハードウェア**:
  - Arduinoボード
  - ボタン（D5に接続）
  - LED（D6に接続）
  - パッシブブザー（D7に接続）
- **ソフトウェア**:
  - Arduino IDE
  - シリアルモニタ

---

## 5. 注意事項
- ボタンの配線は`INPUT_PULLUP`を前提としているため、押下時にGNDに落ちるように接続すること。
- ブザーの音量が小さい場合、外部電源やアンプを使用することを検討。
